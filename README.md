# Swiggy Data Analysis & Optimization Engine (SQL)

## Overview  
This project analyzes Swiggy-like food delivery data using SQL to generate actionable business insights from raw transactional data.

## Objective  
To understand customer behavior, order trends, and restaurant performance for better decision-making.

## Tech Stack  
SQL (MySQL / PostgreSQL)

## Dataset  
Includes data on customers, orders, restaurants, delivery partners, and payments.

## Key Analysis  
- Top restaurants by revenue and orders  
- Customer behavior and repeat patterns  
- Peak order times and demand trends  
- Delivery performance and cancellations  

## SQL Concepts  
Joins, Subqueries, CTEs, Aggregate Functions, Window Functions

## Key Insights  
- Peak hours drive operational strategy  
- High-value customers boost revenue  
- Top restaurants dominate performance  

## Conclusion  
Demonstrates strong SQL and analytical skills by solving real-world business problems using data.
